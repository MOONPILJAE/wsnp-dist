# wsnp-dist

WO(GstarCAD 보일람표 추출 플러그인)의 **자동 업데이트용 배포 저장소**입니다.

- 여기에는 **빌드된 실행파일(DLL)만** 올라갑니다. (소스코드는 비공개 저장소에 별도 보관)
- 사용자는 `WO업데이트.bat` 을 실행하면 이 저장소에서 최신 DLL을 받아 자동 교체합니다.
- 라이선스는 PC마다 별도이며, 업데이트는 라이선스를 건드리지 않습니다.

## 구성
- `version.txt` — 현재 배포 버전 (업데이트 판단 기준)
- `files.txt` — 받을 파일 목록(manifest)
- `*.dll` — 빌드된 플러그인 DLL
- `WO업데이트.bat`, `_wo_update.ps1` — 업데이트 도구(사용자 PC에서 실행)

## 개발자 배포 절차
1. 집(개발) PC에서 WO 빌드
2. 새 DLL을 이 폴더로 복사 + `version.txt` 버전 올림
3. `git add -A && git commit && git push`
4. 사용자/팀원은 `WO업데이트.bat` 더블클릭 → 최신으로
